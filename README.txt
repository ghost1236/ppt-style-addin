PPT 디자인 도구
================================

PowerPoint의 모든 슬라이드에 폰트/색상/스타일을
한 번에 일괄 적용하는 도구입니다.


================================================================
  1. 설치 방법
================================================================

[Mac]
  1. install.command 더블클릭
     (처음 실행 시 우클릭 → "열기" 선택)
  2. PowerPoint가 열리면:
     홈 탭 → "추가 기능" → "디자인 도구" 클릭

[Windows]
  1. install.bat 더블클릭
  2. PowerPoint가 열리면:
     홈 탭 → "추가 기능" → "디자인 도구" 클릭

  한 번 설치하면 이후 PowerPoint를 열 때마다
  추가 기능에 "디자인 도구"가 자동으로 나타납니다.

[제거]
  Mac: uninstall.command 더블클릭
  Win: uninstall.bat 더블클릭


================================================================
  2. 기본 사용법 - 스타일 적용하기
================================================================

  "추가 기능" → "디자인 도구"를 클릭하면
  오른쪽에 사이드 패널이 열립니다.

  1) 적용 대상 선택 (상단 드롭다운)
     ┌─────────────────────────────────┐
     │ 전체 슬라이드 - 모든 텍스트  ▼  │
     └─────────────────────────────────┘
     - 전체 슬라이드 - 모든 텍스트 : 모든 텍스트에 적용
     - 전체 슬라이드 - 제목만     : 제목 영역만 적용
     - 전체 슬라이드 - 본문만     : 본문 영역만 적용
     - 같은 위치 → 전체 슬라이드  : 선택한 도형과 같은
                                     위치의 도형에 적용
     - 선택한 텍스트 상자         : 클릭한 도형에만 적용
     - 선택한 텍스트만            : 드래그한 텍스트만 적용

  2) 스타일 설정
     - 폰트: 드롭다운에서 글꼴 선택
     - 크기: 숫자 입력 (pt 단위)
     - B I U: 굵게/기울임/밑줄 토글
     - 색상: 색상 박스 클릭 → 색상 선택 또는 HEX 입력

  3) 단락 설정
     - 정렬: 왼쪽/가운데/오른쪽/양쪽 정렬
     - 줄간격: 숫자 입력 (1.0 = 1배, 1.5 = 1.5배)

  4) "적용" 버튼 클릭
     → 선택한 대상에 스타일이 일괄 적용됩니다

  5) 실행취소
     → 적용 후 되돌리고 싶으면 ↩ 버튼 클릭
     → 최대 10단계까지 되돌릴 수 있습니다


================================================================
  3. 프리셋 사용법 - 자주 쓰는 스타일 저장하기
================================================================

  프리셋을 저장하면 매번 스타일을 설정하지 않고
  한 번에 적용할 수 있습니다.

  [프리셋 저장하기]
  1) "스타일" 탭에서 원하는 스타일을 설정
  2) "저장" 버튼 클릭
  3) 프리셋 이름 입력 (예: "제목 스타일", "본문 기본")
  4) 저장 완료

  [프리셋 적용하기]
  1) "프리셋" 탭 클릭
  2) 원하는 프리셋의 "적용" 버튼 클릭
  3) 현재 적용 대상에 해당 스타일이 적용됩니다

  [프리셋 수정/삭제]
  - 연필 아이콘: 프리셋 이름/스타일 수정
  - 휴지통 아이콘: 프리셋 삭제


================================================================
  4. 프리셋 맵핑 - 리본 메뉴에 프리셋 연결하기
================================================================

  프리셋을 리본 메뉴 버튼에 연결하면
  사이드 패널을 열지 않고도 한 번에 적용할 수 있습니다.

  [제목용/본문용 프리셋 지정]

  "프리셋" 탭에서 원하는 프리셋을 찾은 후:

  - "제목" 버튼 클릭
    → 이 프리셋이 제목용으로 지정됩니다
    → 리본 메뉴 "일괄 적용 ▼" → "제목 일괄 적용" 클릭 시
      모든 슬라이드의 제목에 이 스타일이 적용됩니다

  - "본문" 버튼 클릭
    → 이 프리셋이 본문용으로 지정됩니다
    → 리본 메뉴 "일괄 적용 ▼" → "본문 일괄 적용" 클릭 시
      모든 슬라이드의 본문에 이 스타일이 적용됩니다

  지정된 프리셋은 버튼이 파란색으로 표시되고
  카드에 [제목] [본문] 배지가 나타납니다.
  다시 클릭하면 지정 해제됩니다.

  [프리셋 슬롯 (P1/P2/P3) 지정]

  자주 쓰는 프리셋을 슬롯에 지정하면
  리본 메뉴에서 바로 적용할 수 있습니다.

  "프리셋" 탭에서 원하는 프리셋을 찾은 후:

  - P1, P2, P3 버튼 중 하나를 클릭
    → 해당 슬롯에 프리셋이 지정됩니다
    → 리본 메뉴 "프리셋 ▼" → "프리셋 1/2/3" 클릭 시
      현재 선택한 도형에 이 스타일이 적용됩니다

  지정된 슬롯은 버튼이 파란색으로 표시되고
  카드에 [P1] [P2] [P3] 배지가 나타납니다.
  다시 클릭하면 지정 해제됩니다.

  [프리셋 맵핑 예시]

  예를 들어, 3개의 프리셋을 만들었다면:

  "제목 스타일"   → "제목" 버튼 클릭 + P1 클릭
  "본문 스타일"   → "본문" 버튼 클릭 + P2 클릭
  "강조 스타일"   → P3 클릭

  이렇게 설정하면 리본 메뉴에서:
  - "일괄 적용 ▼" → "제목 일괄 적용"
    : 모든 슬라이드 제목을 "제목 스타일"로 변경
  - "일괄 적용 ▼" → "본문 일괄 적용"
    : 모든 슬라이드 본문을 "본문 스타일"로 변경
  - "프리셋 ▼" → "프리셋 1"
    : 선택한 도형을 "제목 스타일"로 변경
  - "프리셋 ▼" → "프리셋 3"
    : 선택한 도형을 "강조 스타일"로 변경


================================================================
  5. 같은 위치 → 전체 슬라이드 기능
================================================================

  슬라이드에서 특정 위치의 텍스트 상자를 선택하면
  모든 슬라이드에서 같은 위치에 있는 텍스트에
  스타일을 일괄 적용합니다.

  사용 방법:
  1) 슬라이드에서 텍스트 상자를 클릭하여 선택
  2) 적용 대상을 "같은 위치 → 전체 슬라이드"로 변경
  3) 스타일 설정 후 "적용" 클릭

  예: 모든 슬라이드의 왼쪽 상단 로고 텍스트를
  한 번에 동일한 스타일로 변경할 때 유용합니다.


================================================================
  6. 프리셋 내보내기/불러오기
================================================================

  다른 사람과 프리셋을 공유하거나
  백업할 수 있습니다.

  [내보내기]
  1) "프리셋" 탭 하단의 "JSON 내보내기" 클릭
  2) JSON 파일이 다운로드됩니다

  [불러오기]
  1) "프리셋" 탭 하단의 "불러오기" 클릭
  2) 이전에 내보낸 JSON 파일 선택
  3) 프리셋이 추가됩니다
