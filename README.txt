PPT 디자인 도구 - 설치 안내
================================

PowerPoint에 폰트/색상/스타일을 일괄 적용하는 도구입니다.
설치 파일 하나만 실행하면 자동으로 설치됩니다.


[Mac 설치]
================================

1. install.command 더블클릭
   (처음 실행 시 우클릭 → "열기" 선택)
   → 자동으로 다운로드 및 설치됩니다

2. PowerPoint가 열리면:
   홈 탭 → "추가 기능" 버튼 클릭
   → "디자인 도구" 클릭

   끝! 이후 PowerPoint를 열 때마다
   추가 기능에 "디자인 도구"가 자동으로 나타납니다.


[Windows 설치]
================================

1. install.bat 더블클릭
   → 자동으로 다운로드 및 설치됩니다

2. PowerPoint가 열리면:
   홈 탭 → "추가 기능" 버튼 클릭
   → "디자인 도구" 클릭

   끝! 이후 PowerPoint를 열 때마다
   추가 기능에 "디자인 도구"가 자동으로 나타납니다.


[사용 방법]
================================

디자인 도구를 클릭하면 사이드 패널이 열립니다.

● 스타일 편집 탭
  - 폰트/크기/색상/굵기/기울임/밑줄 설정
  - 정렬/줄간격 설정
  - 적용 대상 선택:
    · 선택한 텍스트만
    · 선택한 텍스트 상자
    · 선택 위치 → 전체 슬라이드
    · 모든 슬라이드 - 제목/본문/전체
  - "적용" 클릭으로 일괄 적용
  - 실행취소 지원 (최대 10단계)

● 저장된 프리셋 탭
  - 자주 쓰는 스타일을 프리셋으로 저장
  - 제목용/본문용 프리셋 지정
  - 프리셋 슬롯 (P1/P2/P3) 지정
  - JSON 내보내기/불러오기

● 리본 메뉴 (홈 탭 → 디자인 도구 그룹)
  - 스타일 패널: 사이드 패널 열기
  - 일괄 적용: 제목/본문 일괄 적용
  - 프리셋: 슬롯 1~3 빠른 적용


[요구 사항]
================================

- Microsoft PowerPoint (데스크톱 버전)
- Microsoft 365 구독
- 인터넷 연결 필요
- Node.js, Python 등 별도 설치 불필요
